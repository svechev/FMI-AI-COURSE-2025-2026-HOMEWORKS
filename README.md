# Searching Algortihms
1. frog-leap-dfs.py: DFS for solving the frog leap problem, frog-leap-linear.py provides a linear solution - a predetermined pattern for the frogs' moves.
2. n-puzzle.py: Uses IDA* for finding a solution for the n-puzzle problem.
3. n-queens.cpp: Uses max-min-conflicts to place N queens on a NxN chess board without conflicts.
4. tsp.py: Uses genetic algorithms to solve the Travelling Salesman Problem.
5. tic-tac-toe: Uses alpha-beta pruning to find the optimal move in tic-tac-toe.

# Machine Learning
6. knn.py: Uses KNN to classify data in the data set iris.data
7. naive-bayes.py: Uses the Naive Bayes algorithm to classify data in the data set house-votes-84.data
8. decision-tree.py: Uses an ID3 tree (with pre-pruning and post-pruning) to classify data in the data set breast-cancer.data
